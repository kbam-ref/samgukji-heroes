# 3D 빌보드 필드 — 설계 (2026-07-20)

## 목표
방어 전장을 Three.js WebGL 씬으로 렌더해 **3D 깊이(원근·조명·그림자)**를 주되,
캐릭터는 **기존 치비 컷아웃 PNG를 빌보드**로 세워 아트·정체성을 100% 유지한다(2.5D).
수석 결정(2026-07-20): 프로토타입 비교 후 A(절차적)·C(이미지→3D) 제치고 **B(빌보드 2.5D)** 채택.

## 원칙 / 불변식
- **엔진·데이터 불변**: `js/systems/defense.js`, `js/data/defense.js`는 수정하지 않는다. 3D는 순수 렌더러.
- **세이브 스키마 무관**: 표현 계층만 바뀌므로 version/마이그레이션 불필요.
- **오프라인 완전**: Three.js는 로컬 번들(`js/vendor/three.module.js`, r160). 외부 요청 0. sw.js ASSETS에 추가.
- **모듈 경계**: 3D 렌더러는 `ui/`에만 존재. 엔진 좌표(x/y 0~100)를 읽기만 한다.

## 구조
```
js/ui/defense-3d.js   (신규)  — Three.js 씬 소유: 카메라·바닥·조명·빌보드 풀·이펙트.
                                init(mountEl) / sync(run, dt) / resize() / dispose()
js/ui/defense-screen.js (수정) — 필드 DOM 렌더를 defense-3d 캔버스로 대체.
                                HUD·독·버튼·시트·이름/별 오버레이는 DOM 유지.
```

### 좌표 매핑
- 엔진: `x∈[0,100], y∈[0,100]` (기존 % 좌표). 바닥 평면 y=0.
- 월드: `worldX = (x-50)/50 * HALF_W`, `worldZ = (y-50)/50 * HALF_D`. 빌보드는 y = 높이/2.
- 카메라: 원근, **적당한 기울기(약 35°)** 로 디오라마 뷰. 라이브에서 각도 미세조정.

### 빌보드
- 유닛/적 = `PlaneGeometry` + 컷아웃 텍스처(`heroes-cut/{id}.png`, `enemies-cut/{art}.png`), `alphaTest`로 외곽 처리.
- **카메라를 바라봄**(Y축 빌보딩 — 수직은 유지해 서 있는 느낌). 이동 방향에 따라 좌우 뒤집기(scale.x ±).
- 발밑 그림자 = 반투명 원 디스크(성능상 실제 그림자 대신 페이크; 캐릭터엔 페이크, 바닥엔 조명 그림자).
- 텍스처는 id별 1회 로드 캐시. 지오메트리는 공유(1개)로 드로우콜 최소화.

### HUD / 텍스트 오버레이
- 이름·별(왕별)·타이머·독·시트는 기존 DOM 유지. 캐릭터 이름표만 매 프레임 `project()`로 화면좌표 갱신해 캐릭터 위에 띄운다(폰트·왕별 연출 그대로 재사용).

## 단계
1. **코어 필드**: 씬·카메라·조명·바닥 + 유닛/적 빌보드를 엔진 상태(생성·이동·제거)에 연결. 배치 드래그 = 레이캐스트로 바닥 히트 → u.tx/ty. 이름표 오버레이.
2. **생명감·이펙트**: idle sway, 타격 스쿼시·플래시, 투사체(화살·칼)·초월 광역기 연출을 3D로.
3. **폴리시**: 배경(3D 스카이/지형), 조명 무드, 성능(100 빌보드 프로파일링), reduce-motion 대응.

## 리스크 / 대응
- **성능(100+ 빌보드)**: 공유 지오메트리 + 텍스처 캐시 + `frustumCulled`. 필요 시 InstancedMesh/아틀라스. 매 프레임 objs 생성 금지(풀 재사용).
- **드래그 히트테스트**: `Raycaster`로 바닥 평면 교차 → 좌표 역매핑. 유닛 선택은 빌보드 레이캐스트.
- **가독성**: 기울기 과하면 원거리 적 축소 → 중간 기울기, 라이브 조정.
- **로딩**: 텍스처 비동기 로드 중 빈 프레임 → 로드 전까지 페이크 그림자만, 로드 후 페이드인.

## 완료 기준(1단계)
- 방어 화면이 3D 캔버스로 뜨고, 유닛/적이 치비 아트로 3D 필드에 서 있으며 엔진대로 이동·전투 위치가 반영된다.
- 배치 드래그·소환·합성이 3D에서 동작. HUD·독·버튼·시트 정상. 콘솔 에러 0. 세이브 정상.
