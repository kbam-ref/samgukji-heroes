# 삼국지 랜덤 디펜스 — 기획안 (v2, 다중 LLM 밸런스 검증 반영)

> 방치+가챠 영웅키우기 → **드래곤볼/메이플 랜덤 디펜스 계열 TD**로 전면 전환.
> 모든 수치는 `js/data/defense.js`에 데이터로 존재. 2026-07-19 다중 검증으로 치명 문제 7건 수정.

## 1. 코어 루프
1. 런 시작(**티켓 −1**) → **무료 10연차 소환**으로 초기 병력.
2. 네모 트랙을 도는 적을 유닛이 **사거리 자동 공격**으로 처치.
3. 처치 골드로 **소환 / 단련 / 도박**, 안 쓰는 유닛은 **반환(판매)**.
4. **같은 등급 3장 → 상위 등급 랜덤 1장**(수동 합성, 등급 올려치기).
5. 스테이지 적 100 처치 시 다음 단계. **50단계 클리어 = 승리**.
6. 살아있는 적 **100 누적 → 게임오버** → 세이브 삭제, **1단계부터 리셋**.
7. 반복 도전 사이 **옥구슬로 영구 성장**을 사서 점점 더 깊이 도달(과금 축).

## 2. 필드·진행
- 사각 트랙, 적 시계방향 순환. 유닛 배치 칸 **16**.
- 총 **50단계**, 단계당 적 100(보스 단계는 **일반 98 + 보스 2**) → 총 5000.
- 적 체력 = `26 × 1.13^(stage-1) × (1+0.01×index) × 체급배수`. (1.13: 50단계 ≈ 399배)
- 살아있는 적 100 누적 = 패배(보스도 각 1로 포함).

## 3. 소환·재화(골드)
- 1소환 = **100골드**. 등급 확률: 전설1%·영웅5%·희귀15%·고급34%·일반45%. **천장: 50뽑마다 등급업 확정**.
- 수입 = **개체당 골드**: `2 × 체급배수 × (보스면 ×8) × 1.15^(stage-1)`. (검증: flat 수입은 후반 붕괴 → 스테이지 스케일)
- 시작 골드 0(초기 병력은 무료 10연차).

## 4. 성장 3축
- **합성(등급 climb)**: 같은 등급 아무 영웅 **3장 → 상위 등급 랜덤 1장**(수동). 전설은 합성 없음. 승급체는 **최고 단련 계승**, 나머지 2장 단련골드 50% 환급.
- **단련**: 골드로 개별 유닛 강화. 비용 40×1.30^lv, 데미지 **+30%/단계**, 최대 20.
- **반환(판매)**: 등급값[50,60,80,95,100] + 단련골드 50% 환급. **등급값은 소환가 이하로 캡**(즉매 차익 차단).

## 5. 상성 2축 (곱연산)
`데미지 = base × 등급배수 × (1 + 0.30×단련lv) × 속성 × 크기 × (1+영구데미지%)`
- **속성(물불땅)**: 물>불>땅>물. 강 **1.5** / 약 **0.75** / 그외 1.0. 영웅 8:8:8, 속성×크기 탈상관.
- **크기(소중대)**: 영웅별 강 **1.25** / 약 **0.8**. 8:8:8.
- 적 체급 분포: 소형 50% / 중형 30% / 대형 20%. 속성은 스폰 시 랜덤.

## 6. 보스
- 매 **5스테이지** 보스 **2마리**(그 단계 100의 일부). 2마리 다 잡아야 클리어.
- 체력 = 그 단계 중형 ×**7**, 속도 ×0.5, 골드 ×8.
- 처치당 무료 소환권 **2 + floor(stage/10)**회 (초반 폭주 완화).

## 7. 도박
- 골드 **100** 소모, 결과: 0(40%)·50(28%)·100(20%)·300(10%)·800(1.5%)·**1500(0.5%)**. 기대값 ≈83.5(RTP 84%). 위로금 밴드 + 15배 잭팟.

## 8. 도전 티켓 & 수익화
- 티켓 시작 5·상한 6·**20분당 1 회복**. **신규 런 시작 시 −1**(이어하기 무료, 세이브스컴 차단). 0이면 신규 런 불가 → 옥구슬 100 충전.
- **영구 성장**(옥구슬): 시작골드+·전유닛데미지%+·오프닝연차+. 리셋형이라 이게 있어야 재도전=진행, 과금이 벽을 민다.
- **옥구슬 획득처**: 보스단계 클리어 +8·최고기록 +30·50단계 최초 +200.

## 9. 저장
- **런 세이브(이어하기·1슬롯)**: 유닛·단계·골드·경과시간. 불러와 이어가기 무료(티켓 소모 없음). **패배 시 삭제 + 1단계 리셋**(추가 페널티 없음, 티켓은 시작 때 이미 차감).
- **메타(영구)**: 티켓·옥구슬·영구성장·최고기록(도달 단계·클리어 시간).

## 10. 랭킹 (추후 서버)
- 2트랙: **미클리어 = 도달 스테이지**, **클리어 = 소요 시간**. 클리어 시간 기록을 처음부터 저장(서버 이전 가능 구조).

## 11. 검증에서 잡은 치명 문제 (반영 완료)
- 수입 flat vs HP 기하 → 경제·진행 붕괴 → **골드 스테이지 스케일**.
- HP 1.22^49≈17,000배(내 초안 오산 '3800배') → **1.13**.
- 리셋형 티켓이라 과금해도 벽 못 뚫음 → **옥구슬 영구성장** 도입.
- 티켓 규칙 모순(세이브스컴·무한리롤) → **런 시작 차감**.
- 같은 영웅 3겹 합성 = 확률상 불가 → **같은 등급 3→상위 랜덤**(장르 표준).
- 자동합성이 단련 파괴 → **계승+환급**. 로스터 속성×크기 상관 언럭 → **탈상관 재배분**.
